# Design and Implementation of an IoT-Based Smart Mobile Animal Housing System

| **4** | **Title of Senior Design Project:** Design and Implementation of an IoT-Based Smart Mobile Animal Housing System |
| **5** | **Design to be Achieved:** This project involves the design and implementation of an IoT-based smart mobile animal housing system that features a floorless structure and is equipped with solar-powered motorized wheels, enabling autonomous relocation to new sites each day with customizable settings. This design ensures that the animals have continuous access to fresh grass and a clean environment, while also preventing manure accumulation and promoting natural physical activity. The system is designed to automatically manage smart lighting, ventilation fans, and environmental conditions within the housing. It can detect the time of day and motion, automatically turning on the lights when movement or human presence is detected. Additionally, the system features a smart entry porch that detects the presence of individuals at the door and prompts a secure password verification for entry. If the correct password is entered, the door unlocks automatically; otherwise, an intrusion alarm is triggered. The system also includes an advanced intrusion detection mechanism, particularly monitoring the roof area, which activates a burglar alert in the event of unauthorized access. An optional automated water management function allows the control of a water pump to refill the tank, with an ultrasonic sensor monitoring the water level in real time. Furthermore, the system integrates environmental sensors, including a temperature sensor, solar intensity tracker, and rain sensor for weather monitoring and adaptive control. All collected data is time-stamped and stored by day and hour, allowing the system to analyze user behavior and operational patterns. This data serves as the foundation for future AI-based recommendation features aimed at optimizing energy efficiency, improving security, and simplifying daily management tasks. The system's overall functionality can be monitored and controlled through a centralized, user-friendly mobile or web application. Based on user analytics, the app may send intelligent notifications, such as: "The lights are usually off at this time. Would you like to turn them off now?" |
| **6** | **Engineering Standards:** WiFi, GPS, and Bluetooth |
| **7** | **Design Requirements:** The design shall achieve the following requirements: 1. The operating voltage for all sensors is between 3 to 5 volts. 2. The sensor's data are sent every 10 minutes (adjustable). 3. The data is stored in time frames and days of the week for the system to learn the environment behavior and user habits. 4. All components should operate within the industrial temperature range -40°C to +85°C. 5. An alert is sent when the water tank level is below 25% of its capacity. 6. An alert is sent when an intruder is inside the coop. 7. The fan is turned on when the temperature is above 30°C and automatically turns off when the temperature is below 29°C. 8. Based on the width of the coop, the coop moves a distance greater than its width by 1 meter (adjustable). 9. Based on the length of the coop, the coop moves a distance greater than its length by 1 meter (adjustable). |
| **8** | **Realistic Constraints:** Economic: Up to 500 JDs. Environmental: Minimum power consumption. Manufacturability and Sustainability: Should be scalable and built using available commercial components. |
